# Robuzta Techlabs - Premium Device Repair Website

A modern, responsive website built for **Robuzta Techlabs**, a premium device repair service provider specializing in laptops, smartphones, tablets, desktops, and gaming PCs.

The website focuses on a clean user experience, smooth animations, responsive design, and interactive components to showcase repair services and customer trust.

---

## 🚀 Features

### Responsive Design

* Fully responsive layout for desktop, tablet, and mobile devices.
* Mobile-friendly hamburger navigation menu.

### Hero Section

* Premium landing section with animated content.
* Clear call-to-action buttons for repair quotes and services.

### Services Section

* Laptop repair services.
* Smartphone and tablet repair services.
* Desktop and gaming PC solutions.

### About Section

* Company introduction.
* Journey timeline.
* Founder information.

### Why Choose Us Section

Includes:

* Free pickup and delivery.
* Fast repairs.
* Genuine spare parts.
* Warranty support.
* Data security.
* Certified technicians.
* Remote assistance.

### Customer Testimonials

* Interactive testimonial slider.
* Previous/next navigation.
* Automatic testimonial rotation.

### FAQ Accordion

* Expand/collapse frequently asked questions.
* Single active question behavior.

### Quote Request Form

* Client information collection.
* Frontend validation.
* Success confirmation modal.

### Animations

* Scroll-based animations using AOS.
* Page-load animations using GSAP.

---

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Libraries

* AOS (Animate On Scroll)
* GSAP Animation Library
* Font Awesome Icons
